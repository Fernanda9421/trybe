# JavaScript - Eventos

Neste módulo, será abordado como utilizar os *eventos do JavaScript* para fazer com que funções sejam executadas quando alguém as usam ou quando o navegador fizer alguma interação com a sua página web.



### Principais pontos abordados:

- [x] Mudar o conteúdo, estilo e outros atributos de elementos HTML a partir de eventos do *JavaScript*
- [x] Aplicar no seu código *JavaScript* eventos de `click` e `change` , entre outros
